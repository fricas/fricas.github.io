EuclideanDomain
===============

:viewcode:`catdef.spad line 413 <catdef.spad#L413>` :editcode:`[edit on github] <catdef.spad#L413>`


A constructive euclidean domain, i.e. one can divide producing a quotient and a remainder where the remainder is either zero or is smaller (\ :spadfun:`euclideanSize`\ ) than the divisor.   Conditional attributes: multiplicativeValuation \ ``Size(a*b)=Size(a)*Size(b)``\  additiveValuation \ ``Size(a*b)=Size(a)+Size(b)``\

:spadfun:`extendedEuclidean`\: (%, %) -> Record(coef1: %, coef2: %, generator: %) 
  \ ``extendedEuclidean(x, y)``\  returns a record rec where \ ``rec.coef1*x+rec.coef2*y = rec.generator``\  and rec.generator is a \ ``gcd``\  of \ ``x``\  and \ ``y``\ . The \ ``gcd``\  is unique only up to associates if \ :ref:`canonicalUnitNormal <l63616e6f6e6963616c556e69744e6f726d616c>`\  is not asserted. \ :spadfun:`principalIdeal`\  provides a version of this operation which accepts an arbitrary length list of arguments.


.. index::
   pair: EuclideanDomain; extendedEuclidean

:spadfun:`extendedEuclidean`\: (%, %, %) -> Union(Record(coef1: %, coef2: %), failed) 
  \ ``extendedEuclidean(x, y, z)``\  either returns a record rec where \ ``rec.coef1*x+rec.coef2*y=z``\  or returns "failed" if \ ``z``\  cannot be expressed as a linear combination of \ ``x``\  and \ ``y``\ .
