FriCAS in COCALC (formerly SageMathCloud)¶

You can try FriCAS commands online in CoCalc.

You have to create a Sage Worksheet. Then prepend before every FriCAS command a line

‘%fricas’

so that your command is not interpreted as a Sage command, but rather as a FriCAS command.
