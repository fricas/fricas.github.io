This package provides functions for integration, limited integration, extended integration and the risch differential equation for elemntary functions.

lfextendedint: (F, Symbol, F) -> Union(Record(ratpart: F, coeff: F), failed)
lfextendedint(f, x, g) returns functions [h, c] such that dh/dx = f - cg, if (h, c) exist, “failed” otherwise.
lfinfieldint: (F, Symbol) -> Union(F, failed)
lfinfieldint(f, x) returns a function g such that dg/dx = f if g exists, “failed” otherwise.
lfintegrate: (F, Symbol) -> IntegrationResult F
lfintegrate(f, x) = g such that dg/dx = f.
lflogint: (F, Symbol, List F) -> Union(Record(mainpart: F, limitedlogs: List Record(coeff: F, logand: F)), failed)
lflimitedint(f, x, [g1, ..., gn]) returns functions [h, [[ci, gi]]] such that the gi's are among [g1, ..., gn], and d(h+sum(ci log(gi)))/dx = f, if possible, “failed” otherwise.
