DistributionContinuedFractionPackage(R, z)¶

distro.spad line 1070 [edit on github]

  • R: CommutativeRing

  • z: Symbol

A package to compute Jacobi continued fractions of Cauchy transforms.

JContinuedFraction: (Distribution R, UnivariatePolynomial(z, Fraction R)) -> ContinuedFraction UnivariatePolynomial(z, Fraction R) if R hasn’t Field and R has IntegralDomain

JContinuedFraction(d, z) returns the Cauchy transform as a continued fraction at z.

JContinuedFraction: (Distribution R, UnivariatePolynomial(z, R)) -> ContinuedFraction UnivariatePolynomial(z, R) if R has Field

JContinuedFraction(d, z) returns the Cauchy transform as a continued fraction at z.
