SupFractionFactorize contains the factor function for univariate polynomials over the quotient field of a ring S such that the package MultivariateFactorize works for S

factor: SparseUnivariatePolynomial Fraction P -> Factored SparseUnivariatePolynomial Fraction P
factor(p) factors the univariate polynomial p with coefficients which are fractions of polynomials over R.
squareFree: SparseUnivariatePolynomial Fraction P -> Factored SparseUnivariatePolynomial Fraction P
squareFree(p) returns the square-free factorization of the univariate polynomial p with coefficients which are fractions of polynomials over R. Each factor has no repeated roots and the factors are pairwise relatively prime.
