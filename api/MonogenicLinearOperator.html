  • R: Ring

This is the category of linear operator rings with one generator. The generator is not named by the category but can always be constructed as monomial(1, 1). For convenience, call the generator G. Then each value is equal to sum(a(i)*G^i, i = 0..n) for some unique n and a(i) in R. Note that multiplication is not necessarily commutative. In fact, if a is in R, it is quite normal to have a*G \~= G*a.
