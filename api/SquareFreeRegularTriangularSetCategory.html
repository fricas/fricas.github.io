The category of square-free regular triangular sets. A regular triangular set ts is square-free if the gcd of any polynomial p in ts and differentiate(p, mvar(p)) with respect to collectUnder(ts, mvar(p)) has degree zero with respect to mvar(p). Thus any square-free regular set defines a tower of square-free simple extensions.
