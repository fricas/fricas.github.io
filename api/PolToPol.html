Package with the conversion functions among different kind of polynomials

dmpToHdmp: DistributedMultivariatePolynomial(lv, R) -> HomogeneousDistributedMultivariatePolynomial(lv, R)

dmpToHdmp(p) converts p from a DMP to a HDMP.

dmpToP: DistributedMultivariatePolynomial(lv, R) -> Polynomial R

dmpToP(p) converts p from a DMP to a POLY.

hdmpToDmp: HomogeneousDistributedMultivariatePolynomial(lv, R) -> DistributedMultivariatePolynomial(lv, R)

hdmpToDmp(p) converts p from a HDMP to a DMP.

hdmpToP: HomogeneousDistributedMultivariatePolynomial(lv, R) -> Polynomial R

hdmpToP(p) converts p from a HDMP to a POLY.

pToDmp: Polynomial R -> DistributedMultivariatePolynomial(lv, R)

pToDmp(p) converts p from a POLY to a DMP.

pToHdmp: Polynomial R -> HomogeneousDistributedMultivariatePolynomial(lv, R)

pToHdmp(p) converts p from a POLY to a HDMP.
