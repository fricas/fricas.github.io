RationalFunctionIntegration F¶

intrf.spad line 594

  • F: Join(IntegralDomain, RetractableTo Integer, CharacteristicZero)

This package provides functions for the integration of rational functions.

infieldIntegrate: (Fraction Polynomial F, Symbol) -> Union(Fraction Polynomial F, failed)
infieldIntegrate(f, x) returns a fraction g such that dg/dx = f if g exists, “failed” otherwise.
internalIntegrate: (Fraction Polynomial F, Symbol) -> IntegrationResult Fraction Polynomial F
internalIntegrate(f, x) returns g such that dg/dx = f.
